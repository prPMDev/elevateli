🔒 Your Privacy is Our Priority

ElevateLI operates entirely in your browser. We do not collect, store, or transmit any of your LinkedIn data to our servers because we don't have any servers. Your profile information never leaves your device.

What We Do (and Don't) Collect

✅ What Stays on Your Device

  • LinkedIn Profile Data: All profile information analyzed by the extension remains locally in your browser
  • Analysis Results: Completeness scores and AI feedback are stored only in your browser's local storage
  • API Keys: Your OpenAI/Anthropic API keys are encrypted and stored locally in your browser
  • Settings: Your preferences and configuration are saved locally

❌ What We Never Collect

  • Your LinkedIn profile information
  • Names, job titles, or any personal details
  • Your connections or network information
  • Usage analytics or tracking data
  • Your API keys or AI responses

How Your Data is Processed

Local Analysis

The completeness scoring happens entirely in your browser using JavaScript. No external servers are involved in calculating your profile score.

AI Analysis (Optional)

If you choose to enable AI-powered content analysis:

  • Your profile data is sent directly from your browser to your chosen AI provider (OpenAI or Anthropic)
  • We never see or intercept this communication
  • The API request uses your personal API key
  • You're bound by your AI provider's privacy policy and terms

Data Storage

Browser Local Storage

We use Chrome's local storage API to save:

  • Cached analysis results (to avoid re-analyzing)
  • Your encrypted API keys
  • Extension settings and preferences

This data is:

  • Stored only on your device
  • Never transmitted to external servers
  • Cleared when you uninstall the extension
  • Can be manually cleared at any time through the extension settings

Third-Party Services

LinkedIn

The extension reads data from LinkedIn pages you visit. We don't modify or interfere with LinkedIn's own data collection practices.

AI Providers (Optional)

If you enable AI analysis, you're directly communicating with:

  • OpenAI: OpenAI Privacy Policy
  • Anthropic: Anthropic Privacy Policy

Security Measures

  • Encryption: API keys are encrypted using Web Crypto API before storage
  • No External Communication: The extension doesn't communicate with any servers we control
  • Open Source: Our code is reviewable for security auditing
  • Minimal Permissions: We only request necessary Chrome permissions

Your Rights and Controls

You have complete control over your data:

  • Clear Cache: Remove all stored analysis results anytime
  • Delete API Keys: Remove stored API keys through settings
  • Uninstall: Removes all extension data from your browser
  • Disable AI: Use the extension without any external API calls

Children's Privacy

ElevateLI is intended for professional use by individuals 16 years or older. We do not knowingly collect information from children.

Changes to This Policy

We'll update this privacy policy as needed and notify users of significant changes through the extension update notes.
